
# Resident Monthly Balance Auto-Charge Backend

## Setup Instructions

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Firebase Setup**
   - Set up Firebase credentials and place the key file in a safe location.
   - Set the environment variable for Firebase credentials or provide them through `applicationDefault()`.

3. **Run Locally**
   ```bash
   npm start
   ```

4. **Deploy on Render**
   - Create a Render account and link your repository.
   - Deploy as a web service and enable auto-deployment.
